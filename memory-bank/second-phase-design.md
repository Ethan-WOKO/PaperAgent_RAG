## 1. 第二期目标

第二期定位为「**面向 LaTeX 论文的、有据可查的 AI 审稿/改稿 + 文献补全助手**」。输入输出绑定 LaTeX（`.tex` + 本地 `.bib`），产出可直接在 VSCode 编译。原 docx 路线作废。

当前第二期主线聚焦：

1. LaTeX 解析与文档模型构建（L0），章节角色识别 + 结构确认检查点。
2. 结构化研究画像（L3 种子）。
3. 多源文献检索地基（L3）：去重缓存、读摘要、概念阶梯。
4. gap 分析与有据可查的研究级建议（L4）。
5. 占位保护分章润色（L1）与纯静态结构审查（L2）。
6. 三件套产出：润色后 `.tex` + 审查报告（每条挂真实引用）+ suggested.bib。
7. 论文页：在线预览 + 逐条采纳 + 结构确认交互。

GitHub MCP 与 CLI 已有代码保留，但本期不作为交付门禁，统一放入后续拓展池。

## 3. 第二期不做事项

- 第一版不真编译（不装 TeX），只做静态结构分析；真编译 latexmk 后置。
- 第一版只覆盖结构/实验/相关工作缺口，不碰方法对不对/创新性够不够。
- 进阶版第一版只支持单文件改写；多文件写回（需细粒度 source map）后置。
- 不做多租户组织管理、管理员后台、复杂权限树。
- 不做在线 Skill 编辑器；仍以 `skills/user/` 文件夹方式管理。
- 不追求达到发表级自动化，只追求有据可查、可解释、可评价。
- 不把遗留项目作为运行时依赖，不修改 `PaiSmart-main/`、`paper-agent/`。
- 第二期不强制完成 GitHub MCP 真实联调与 CLI 命令化入口。

## 4. 关键产品改进

### 4.1 LaTeX 审稿/改稿（核心）

- 输入 `.tex` + `.bib`（无 `.bib` 时由文献系统反向生成 suggested.bib）。
- 占位保护机制：受保护元素（数学/`\cite`/`\ref`/`\label`/图表等）mask 后润色，返回校验占位符集合 ⊆ 输入再换回；正文可大改但受保护元素只能保留/移动。
- 允许严格审查前提下的实质性结构修改（调结构/补论证/重组），不只改措辞。
- **防幻觉铁律**：推荐论文必须来自检索 API 真实返回，模型只能从候选选并解释。

### 4.2 文献地基与研究级建议（卖点）

- 多源检索（OpenAlex + arXiv，S2 可插拔），只读摘要，identity 去重 + 三层缓存。
- 概念阶梯（优势组/劣势组各挂文献）+ gap 对比矩阵。
- 立场区分：引言/相关工作站用户这边（反衬），审稿意见站审稿人这边（挑刺）。
- 诚实闸门：吹不动就转批评，绝不编造别人局限。
- 文献检索作为可复用内置能力，对话与论文模块共用 `LiteratureService`。

### 4.3 论文页体验

- 在线预览：源码级高亮 diff + 逐条勾选采纳（非渲染 PDF）。
- 两个入口：基础版（只推荐）/进阶版（改原文 + 自动补 `\cite`）。
- 结构确认检查点交互：批量问、默认保持原样、全部保持一键、角色可改。
- 审查报告（挂真实引用 + 免责声明）、suggested.bib 展示、三件套下载。
- 新增内容诚实分级：A 类真内容、B 类骨架 + 占位 + 理由。

### 4.4 部署化（后置，不阻塞）

- 保留 `docs/DEPLOYMENT.md` 草案；服务器部署、Nginx、HTTPS、备份恢复不进入论文专项门禁。

### 4.5 Skills / MCP / CLI（后续拓展）

- Filesystem MCP 与 code-review 已有能力保留，不作为论文专项阻塞项。

## 5. 风险与控制

| 风险 | 控制方式 |
|---|---|
| 论文质量主观性强 | 固定样例集 + 人工评分维度 + 输出对比记录 |
| 模型幻觉编造引用 | 防幻觉铁律 + evidence 必挂真实卡片 + grounding 校验 cite key |
| 角色识别一步错步步错 | 三级识别 + 内容位置判断 + 结构确认检查点 + 用户可改 |
| 润色破坏可编译性 | 占位保护 mask + 采纳后静态 lint + 掉占位宁可不改 |
| LaTeX 格式复杂 | 务实分词 + 边界规则（verbatim 不透明、自定义命令不支持）+ 不真编译 |
| 长流水线中断 | 可恢复阶段步 + WAITING_INPUT 释放线程 + current_stage 续跑 |
| 文献源网络不稳定 | Mock 测试覆盖解析；真实调用失败不阻塞主结果 |
| 外部模型成本 | analysis 永久缓存复用；manual 与真实外网测试分离 |
